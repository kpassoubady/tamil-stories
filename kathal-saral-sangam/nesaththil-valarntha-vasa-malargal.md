# *நேசத்தில் வளர்ந்த வாசமலர்கள் (காதல் கவிதை தொகுப்பு)*

என் பெயர் காங்கேயன் பசுபதி. எழுத்துலகில் 'கங்கா' என்ற புனைப் பெயரில் எழுதி வருகிறேன். நான் பாண்டிச்சேரியில் பிறந்து வளர்ந்தவன். இப்போது வட அமெரிக்காவில் கலிபோர்னியாவில் உள்ள வளைகுடாப் பகுதியில் வசித்து வருகிறேன். கவிதைகளும், சிறுகதைகளும் எழுதுவது என் மனதிற்கு நெருக்கமான ஒன்று.

காதல் என்பது வெறும் உணர்வின் அலை அல்ல; அது காலத்தையே சுருக்கிடும் கருந்துளை, உயிரியல் மரபணு வரிசைகளை மாற்றும் உணர்வு, பிரிவின் விரிவைச் சுருக்கும் புவிஈர்ப்பு, மெய்த்தொடுதலில் மின்னும் வேதிவினை, கனவின் இருளை வளமாக்கும் ஒளிச்சேர்க்கை.

## *1. காலத்தின் கருந்துளை*

```
உன் முதல் பார்வை விழுந்ததில் —
என் விழி விண்ணின் ஒரு ஒளி;
உன் ஒற்றை நொடி போதும் —
என் நேர சக்கரங்கள் நின்றன!

உன் கண்களின் கருந்துளை —
ஒளியை விழுங்கும் ஆழக் கிணறு;
அதில் விழுந்தேன் நான் —
எப்போதும் அழியாத கவிதையாய் உயிர்த்தேன்!

உன் நாடித் துடிப்புகள் —
என் குருதி நதியில் மின்னல் மீன்கள்;
உன் அருகாமை —
என் மூச்சில் விண்மீன் படிந்த துகள்.
```

## *2. உயிரின் வரைபடம்*

```
என் இதயம் —
அடையாளம் இழந்த வெற்றுப் நிலப்படம்;
நீ கால் பதித்தாய் —
வெறுந்தரை எங்கும் பசுமையானது!

உன் பெயரின் எழுத்துக்கள் —
என் மரபணு வரிசையின் புது வித்துக்கள்;
அதை உச்சரிக்கும் போதெல்லாம் —
என் உயிரணுக்கள் பாட்டுப் பாடுகின்றன!

நாம் பேசாத இடைவெளிகள் —
விரலிடுக்கில் புகுந்து விளையாடும் காற்று;
அந்தத் தீண்டாக் கணங்களிலும் —
என் நரம்புகள் மின்கடத்தியாய் ஒளிர்கிறது.
```

## *3. பிரிவின் புவிஈர்ப்பு*

```
நீ இல்லாத பொழுதுகள் —
முட்கள் உதிர்ந்த சுவர்க் கடிகாரம்;
மணிகள் நகராமல் —
என் ஏக்கம் மட்டும் கேட்கிறது!

உன் நினைவு என்றும் —
பாலை வனத்தில் மறையா கானல் நீர்;
என் இமையை மூடினால் —
உன் முகம் அகத்திரையில் விரிகிறது!

என் தனிமை —
உன் குரலின் எதிரொலி தேடும் கிணறு;
ஒவ்வொரு துளியிலும் —
உன் சிரிப்பு உறைந்த படிகங்கள்.
```

## *4. தீண்டலின் வேதியியல்*

```
உன் விரல் நுனிகள் —
என் மேனியில் ஊரும் மின்சார எறும்புகள்;
நீ தொட்ட இடமெல்லாம் —
என் மெய்சிலிர்ப்பின் பெரும் முழக்கம்!

நம் மூச்சுக் காற்று —
ஒன்றாய் தழுவிய காற்றுப் பாம்புகள்;
உடல்கள் உராயும் போது —
மனம் மின்மினிப் பூக்களாய் மலரும்!

கண்களால் அருந்தும் காதல் —
இதழ்கள் தீண்டாத செந்நிற மது;
என் குருதி ஓட்டத்தில் —
உன் பெயர் நெபுலாவாய் ஓடுகிறது.
```

## *5. கனவின் ஒளிச்சேர்க்கை*

```
நம் இரவு என்பது —
நாம் கைகோர்க்கும் கனவின் காப்பறை;
வானத்து விண்மீன்கள் —
நமக்காய் கண் சிமிட்டும் காவலர்கள்!

விடியல் வந்து பிரித்தாலும் —
உன் புன்முறுவலில் கண்ட ஒளிச்சேர்க்கை;
என் பகல் முழுவதையும் —
உன் நினைவு பசுங்கதிராய் படர்கிறது!.

மறக்க முடியாத நம் காதல் —
எலும்பில் பொறித்த பசுமைச் சின்னம்;
நாம் மறைந்த பின்னரும் அது —
ஒளியாண்டுகள் தாண்டியும் பயணிக்கும்.
```

*கவிஞர் - கங்கா (காங்கேயன் பசுபதி)*
