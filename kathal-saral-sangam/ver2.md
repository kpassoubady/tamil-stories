# *நேசத்தில் வளர்ந்த வாசமலர்கள் (காதல் கவிதை தொகுப்பு)*

என் பெயர் காங்கேயன் பசுபதி. எழுத்துலகில் 'கங்கா' என்ற புனைப் பெயரில் எழுதி வருகிறேன். நான் பாண்டிச்சேரியில் பிறந்து வளர்ந்தவன். இப்போது வட அமெரிக்காவில் கலிபோர்னியாவில் உள்ள வளைகுடாப் பகுதியில் வசித்து வருகிறேன். கவிதைகளும், சிறுகதைகளும் எழுதுவது என் மனதிற்கு நெருக்கமான ஒன்று.

காதல் என்பது வெறும் உணர்வின் அலை அல்ல; அது காலத்தையே சுருக்கிடும் கருந்துளை, உயிரியல் மரபணு வரிசைகளை மாற்றும் உணர்வு, பிரிவின் விரிவைச் சுருக்கும் புவிஈர்ப்பு, மெய்த்தொடுதலில் மின்னும் வேதிவினை, கனவின் இருளை வளமாக்கும் ஒளிச்சேர்க்கை. இந்தக் கருத்துக்களைக் கவிதைகளாக வடித்துள்ளேன். படித்து மகிழுங்கள்.

## *1. காலத்தின் கருந்துளை*

```
உன் முதல் பார்வை —
என் விழி விண்ணின் ஒரு ஒளி;
உன் ஒற்றை நொடி —
காலச் சக்கரங்களை நிறுத்தியது!

உன் கண்களின் கருந்துளை —
ஒளியை விழுங்கும் ஆழக் கிணறு;
அதில் விழுந்தேன் —
அழியாத கவிதையாய்!

உன் நாடியொலி —
என் குருதியில் மோர்ஸ்குறி;
உன் அருகாமை —
என் மூச்சில் விண்மீன்துகள்.
```

## *2. உயிரின் வரைபடம்*

```
என் இதயம் —
அடையாளம் இழந்த வெற்றுநிலப்படம்;
நீ கால் பதித்தாய் —
வெறுந்தரை பசுமையானது!

உன் பெயரெழுத்து —
என் மரபணுவரிசையின் புது மெட்டு;
அதை உச்சரிக்கும் போதெல்லாம் —
என் உயிரணுக்களில் புது இசை!

நம் பேசா இடைவெளி —
விரலிடுக்கில் விளையாடும் காற்று;
அந்தத் தீண்டாக் கணங்களிலும் —
என் நரம்புகள் மின்கடத்தி.
```

## *3. பிரிவின் புவிஈர்ப்பு*

```
உன் வெறுமை —
விண்வெளியின் பூஜ்ய ஈர்ப்பு!
பிரியும் நேரம் —
கடிகார முட்களில் சிக்கிய கைதி!

உன் நினைவு —
பாலைவனத்தில் மறையா கானல்நீர்;
இமையை மூடினால் —
உன் முகம் அகத்திரையில்!

என் தனிமை —
உன் குரலின் எதிரொலி தேடும் கிணறு;
ஒவ்வொரு துளியிலும் —
உன் சிரிப்பு புதைபடிமம்.
```

## *4. தீண்டலின் வேதியியல்*

```
உன் விரல் நுனிகள் —
என் மேனியில் ஊரும் மின்சாரம்;
நீ தொட்ட இடமெல்லாம் —
மெய்சிலிர்ப்பின் முழக்கம்!

நம் மூச்சுக் காற்று —
தழுவிய காற்றுப்பாம்புகள்;
உடல்கள் உராயும் போது —
மனம் மின்மினிப்பூக்கள்!

கண்களால் அருந்தும் காதல் —
இதழ்கள் தொடா செந்நிறமது;
என் குருதி ஓட்டத்தில் —
உன் பெயர் எரிமலைக்குழம்பு.
```

## *5. கனவின் ஒளிச்சேர்க்கை*

```
நம் இரவு —
கைகோர்க்கும் கனவின் காப்பறை;
வானத்து விண்மீன்கள் —
நமக்காய் கண்சிமிட்டும் காவலர்!

விடியல் பிரித்தாலும் —
உன் புன்முறுவலில் கண்ட ஒளிச்சேர்க்கை;
என் பகல் முழுவதும் —
உன் நினைவு பசுங்கதிர்.

மறக்க முடியாத நம் காதல் —
எலும்பில் பொறித்த பச்சைக்குத்தல்;
நாம் மறைந்த பின்னரும் அது —
ஒளியாண்டுகள் தாண்டியும் பயணிக்கும்.
```

*கவிஞர் - கங்கா (காங்கேயன் பசுபதி)*
